WarcraftXL

Warcraft eXtension Layer

Home Wiki Catalog GitHub

WarcraftXL is an interoperability project. It distributes no Blizzard code and no game assets, and runs only against a client you supply and own, reading that client's own files at runtime. Reverse-engineering is limited to what is necessary for interoperability.
